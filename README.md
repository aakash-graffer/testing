## Installation
***
A little intro about the installation. 
```
$ git clone git@bitbucket.org:rupyo/rup-api.git
$ npm install
$ npm install pm2
$ npx secure-env .env -s secret@123
$ node seeder/rupyoadminSeeder.js
$ pm2 start server.js
```
